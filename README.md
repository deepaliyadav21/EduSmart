"# EduSmart"   
I have built a complete LMS (Learning Management System) website using MERN Stack with an AI-powered Quiz Generator and an interactive Dashboard.

This LMS platform allows users to:  
✔️ Learn through structured courses  
✔️ Attempt AI-generated quizzes  
✔️ Track progress via a dashboard  
✔️ Improve knowledge using smart assessments

🛠️ Tech Stack Used:  
MongoDB  
Express.js  
React.js  
Node.js  
AI-based Quiz Generation  
Modern Dashboard UI

🔥 Key Features:
AI Generated Quizzes
User Authentication
Course Management System
Performance Tracking Dashboard
Real-world MERN Project Architecture
